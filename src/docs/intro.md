## Key Features

### Hazard Information
Maintain a comprehensive registry of all identified hazards with:
- Severity and likelihood assessments
- Mitigation status tracking
- Ownership assignment
- Detailed descriptions

### Fault Tree Analysis
Interactive fault trees that show:
- Top events, intermediate events, and basic events
- AND/OR logical gates
- Probability calculations
- Hierarchical relationships

### Safety Performance Indicators
Real-time dashboards displaying:
- Total hazards and mitigation status
- Incident trends over time
- Hazards categorized by type
- Safety metrics completion rates

### Goal Structured Notation
Build structured safety arguments with:
- Goals, strategies, and solutions
- Context and assumption nodes
- Progress tracking
- Evidence linking
